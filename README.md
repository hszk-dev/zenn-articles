# Zenn Articles

Zenn記事管理用リポジトリ

## セットアップ

```bash
npm install
```

## 使い方

```bash
# プレビュー
npm run preview

# 新規記事作成
npm run new:article

# 新規本作成
npm run new:book
```

## ディレクトリ構成

```
.
├── articles/     # 記事
├── books/        # 本
└── images/       # 画像
```
